# NayePankh Volunteer Management System

## Overview

The NayePankh Volunteer Management System is a web-based application developed to help manage volunteer registrations and administrative activities efficiently.

## Features

* Volunteer Registration Form
* AI Assistant Chatbot
* Dark Mode
* Admin Login Page
* Admin Dashboard
* Volunteer Data Management
* Backend API using Node.js and Express
* Responsive User Interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* CORS

## Project Purpose

The project aims to simplify volunteer management by providing an easy-to-use registration system and dashboard for administrators.

## Future Improvements

* Database Integration
* Email Notifications
* Advanced AI Chatbot
* Authentication System
* Volunteer Analytics Dashboard
